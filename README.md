# TechNova Sales Analysis using SQL

## 📌 Project Overview
This project analyzes TechNova Electronics sales data using MySQL Workbench. The objective is to extract meaningful business insights by applying SQL queries to sales data. The project demonstrates fundamental and intermediate SQL concepts commonly used by Data Analysts and Business Analysts.

## 🎯 Objectives
- Analyze sales performance.
- Generate branch-wise and monthly sales reports.
- Identify top-performing salespersons.
- Analyze payment mode usage.
- Create reusable SQL reports using views.
- Perform data analysis using SQL queries.
